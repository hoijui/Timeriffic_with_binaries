按照您的日程開啟或關閉靜音和振動.
簡單. 易用. Timeriffic.
沒有多餘的功能.
不會損壞您的電池!

例如：上午8點有課(抱歉此範例對您可能不合適), 使用Timeriffic來:

  • 在上午7點到8點騎自行車去教室途中將音量設定成響鈴,
  • 在上午8點切換成振動模式,
  • 然後在上午9點切換迴響鈴.
(如果你是一個成年人，將上課替換成上班).

新特性

 Version 1.09 adds the following new features:
  • Detect and support "Notification uses Ring Volume".
  • Detect and support auto-brightness.
Version 1.8 adds the following new features:
  • New menu for error reporting or request help.
  • Translated in Spanish.
  • Fixed crash when inserting more than 15 actions in a row.
  • Fixed settings were being reset each time there was an incoming phone call.
  • Fixed crash when switching to landscape in edit action percentage dialog.
  • New option to turn vibrate notification on but vibrate ringer off.
  • Fixed a couple reported crashes.
  • Translated in Korean by Ubinuri (fingertool.com).
  • Fix: correctly support manual time changes and daylight time change.
  • Roundup brightness to nearest 5%.
Version 1.7 adds the following new features:
  • APNDroid integration. Requires APNDroid 2.0 from market.
  • Correctly recompute alarms when time zone changes.
Version 1.6 adds the following new features:
  • UI changes.
  • Separate notification volume control (for Android 1.5 and up).
  • Bluetooth toggle setting (for Android 2.0 and up).
  • Vibrate now has 2 modes: turn off both ringer & notification vibration (the default) or turn off ringer vibration only (which keep notification vibration on).
Version 1.5 adds the following new features:
  • Wifi toggle setting.
  • Airplane mode toggle setting.
Version 1.4 adds the following new feature:
  • Global brightness setting.
Version 1.3 adds the following new features:
  • Profiles let you group actions together and enable or disable them at once.
  • Actions can be set for given days, such as actions for the week days and others for the week-end.
A couple of default profiles are provided to get you started.

To edit a profile, use long press on a profile item or an action item. A menu will appear to let you edit, insert or delete the item.

已知問題
  • In Donut and Eclair, the way Timeriffic works makes the screen flicker for a short period of time -- it gets darker and then goes back to the requested brightness. This is due to the way Timeriffic works and is not an issue with your phone.
  • Some phones (Droid, HeroC) have a new feature called "Auto Brightness" accessible from Settings > Sound & Display > Brightness. When auto-brightness is enabled, Timeriffic will have no effect and the brightness will not change.
