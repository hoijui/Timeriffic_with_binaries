Du kannst hier einen Fehler von Timeriffic per E-Mail dem Autor melden.

Wenn du den Schalter hier drunter drückst wird eine Mail mit diesen Inhalten generiert:

  • Die letzten Java Ausnahmefehler (wie z.B. 'force closes').
  • Die letzten Aktionen die Timeriffic durchgeführt hat.
  • Eine Zusammenfassung deiner Profile.
  • Der Name deines Android-Handys.
Keine persönliche Identifikation wird verschickt
Die Mail wird in deiner Standard E-Mail Applikation geöffnet, in der du die Mail vor dem Senden anschauen und bearbeiten kannst.
